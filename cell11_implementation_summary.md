## 📋 實作內容

### 核心功能
1. **鄉鎮邊界載入**: 自動載入 TOWN_MOI.shp 並轉換座標系統
2. **區域統計計算**: 使用 rasterstats 計算 Kriging 和 RF 柵格的統計量
3. **決策表格建立**: 產生包含所有必要欄位的 DataFrame
4. **可信度分類**: 基於變異數百分位數分類可信度等級
5. **結果分析**: 識別關鍵組合並提供決策支援

### 輸出格式
```

羅東鎮  宜蘭縣      14.3       42.7    13.8        0.55 MEDIUM
```
